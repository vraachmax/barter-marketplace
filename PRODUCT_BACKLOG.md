## СДЕЛАНО

### Инфраструктура
- [x] Monorepo: NestJS 11 + Next.js 16 + Prisma 6 + PostgreSQL
- [x] JWT auth (AuthProvider + localStorage + Bearer)
- [x] SEO, 404, error boundary, loading skeletons, PWA manifest
- [x] shadcn/ui: 18 компонентов установлены

### Объявления
- [x] CRUD + фильтры + ранжирование + промо TOP/VIP/XL + VIP-полоса
- [x] Геопоиск Haversine + сортировка nearby
- [x] Типы цен, бейдж фото, timeAgo, пагинация infinite scroll
- [x] Fullscreen галерея, breadcrumbs, поделиться, пожаловаться
- [x] Показать телефон с маскировкой

### Карта
- [x] Яндекс.Карты с маркерами цен + карточка при клике
- [x] Мини-карта на объявлении

### Чат
- [x] Socket.IO realtime + медиа + typing + read receipts + presence
- [x] Мини-чат виджет с анимацией + бейдж непрочитанных

### Поиск
- [x] Подсказки с категориями + история запросов
- [x] Синонимы + typo tolerance (Meilisearch)

### Профиль
- [x] Кабинет продавца, настройки, отзывы, публичный профиль
- [x] Смена пароля (API)
