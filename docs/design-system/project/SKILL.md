## Quick reference

- **Primary blue:** `#007AFF` (hover `#0066DD`, soft `#E8F2FF`)
- **Promo accent:** `#FF6F00` (VIP/TOP badges + ring)
- **Page bg:** `#F2F3F5` · **Card:** `#FFFFFF` · **Radius:** 12–16px
- **Font:** Inter (all weights 400–800); Russian-first copy
- **Icons:** lucide-react, strokeWidth 1.8, 16–24px
- **Categories:** each has a 135° two-stop gradient (see `colors_and_type.css`)
