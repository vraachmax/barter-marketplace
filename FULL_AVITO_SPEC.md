# Barter — Полная спецификация функционала (уровень Авито)

## Миссия
Полноценная замена Авито с ценами в 2 раза ниже для бизнеса и копеечными для физлиц.
Размещение объявлений БЕСПЛАТНО (пока не наберётся аудитория — навсегда для физлиц).

## 2. СЕРВИСЫ ПЛАТФОРМЫ

### 2.1 Кошелёк
- Пополнение баланса (карта, СБП)
- Автопополнение при достижении минимума
- История транзакций
- Вывод средств (для продавцов при безопасной сделке)

### 2.2 Безопасная сделка
- Онлайн-оплата через площадку
- Деньги замораживаются до подтверждения получения
- Возврат при проблемах
- Комиссия: 2% (минимум 29 р)

### 2.3 Доставка
- Интеграция с логистическими партнёрами
- Расчёт стоимости по весу/габаритам
- Отслеживание посылки
- Самовывоз с онлайн-оплатой

### 2.4 Верификация
- По паспорту (через партнёра)
- По ИНН (для бизнеса)
- Бейдж "Документы проверены" на профиле
- Приоритет в ранжировании для верифицированных

### 2.5 Бот поддержки (AI-ассистент)
- FAQ по платформе
- Помощь с объявлениями (как продать быстрее, советы по фото/описанию)
- Быстрые ответы продавцу:
  - "Здравствуйте!"
  - "Когда можно посмотреть?"
  - "Сегодня можно посмотреть?"
  - "Завтра можно посмотреть?"
  - "Ещё продаётся?"
  - "Возможен ли торг?"
- Автоответы на типовые вопросы (настраиваемые продавцом)
- Жалобы и споры через бота
- Контекст объявления в диалоге (бот знает о чём речь)

### 2.6 Уведомления
- In-app (колокольчик в шапке):
  - Новое сообщение
  - Ответ на объявление
  - Изменение цены в избранном
  - Объявление снято с публикации
  - Новый отзыв
  - Промо-акция заканчивается
- Email (настраиваемые):
  - Дайджест новых объявлений по сохранённому поиску
  - Снижение цены в избранном
  - Непрочитанные сообщения (через 30 мин)
- Push (Service Worker):
  - Новое сообщение (мгновенно)
  - Ответ покупателя

### 2.7 Рассылка спецпредложений
- Продавец может разослать скидку тем, у кого объявление в избранном
- Шаблоны: "Скидка 10%", "Снижена цена", "Последний экземпляр"

## 3. ВЕРТИКАЛЬНЫЕ КАТАЛОГИ

### 3.1 Авто (Транспорт)
Фильтры:
- Тип (легковые, грузовые, мото, спецтехника)
- Марка / Модель
- Год выпуска (от / до)
- Пробег (от / до)
- Коробка передач (механика, автомат, вариатор, робот)
- Привод (передний, задний, полный)
- Тип двигателя (бензин, дизель, гибрид, электро, газ)
- Объём двигателя (от / до)
- Мощность (от / до)
- Расход топлива
- Кузов (седан, хэтчбек, универсал, внедорожник, минивэн, купе, кабриолет, пикап, фургон)
- Цвет
- Состояние (новый, с пробегом)
- Владельцев по ПТС
- Руль (левый, правый)
- VIN проверка

### 3.2 Недвижимость
Фильтры:
- Тип (квартира, комната, дом, участок, коммерция)
- Сделка (купить, снять надолго, посуточно)
- Комнат (студия, 1, 2, 3, 4+)
- Площадь общая / кухня / жилая (от / до)
- Этаж / этажность (от / до)
- Ремонт (без ремонта, косметический, евро, дизайнерский)
- Балкон (есть / нет / лоджия)
- Санузел (совмещённый, раздельный)
- Мебель (да / частично / нет)
- Техника (перечень)
- Тип дома (панель, кирпич, монолит, дерево)
- Год постройки
- Парковка
- Проверка в Росреестре (собственников, обременения)

### 3.3 Работа
Фильтры:
- Тип (вакансия, резюме)
- Специализация
- Зарплата (от / до)
- Опыт (без опыта, 1-3 года, 3-6 лет, 6+ лет)
- График (полный день, сменный, гибкий, удалённо, вахта)
- Тип занятости (полная, частичная, стажировка, проектная)

### 3.4 Услуги
Фильтры:
- Категория услуги
- Цена (от / до)
- Выезд на дом (да / нет)
- Опыт исполнителя
- Портфолио (есть / нет)
